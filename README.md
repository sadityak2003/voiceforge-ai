# 🎙️ VoiceForge AI

An AI-powered Text-to-Speech SaaS platform built with Spring Boot Microservices, React, PostgreSQL, Docker, and ElevenLabs API.

## 🚀 Tech Stack

### Backend
- Java 21
- Spring Boot 3
- Spring Cloud Gateway
- Eureka Server
- Spring Security + JWT
- OpenFeign
- PostgreSQL
- Docker
- Swagger/OpenAPI

### Frontend
- React
- Vite
- Material UI

### AI
- ElevenLabs API

### Cloud
- Docker
- Google Cloud Platform
